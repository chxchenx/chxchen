Goodreads

This project is unaffiliated with Goodreads. Goodreads is a social cataloging website that allows users to search and sort literary works. This is a mobile redesign of the Goodreads app, focusing on modern aesthetics and more user-friendly functionality.

Using reviews on the app store, I determined that user satisfaction with Goodreads could be improved by addressing the following concerns:
  • design caused the app to be difficult to navigate
  • difficult to connect with friends using app features
  • lack of aesthetics
By implementing modern aesthetics and making the app similar to other social media, users will have an easier time using the app.
